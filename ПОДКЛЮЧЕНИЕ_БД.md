# Подключение SQLite базы данных

## Что было сделано:

1. ✅ Установлены пакеты Entity Framework Core для SQLite
2. ✅ Настроено подключение к базе данных `C:\Users\Женя\PycharmProjects\pythonProject2\проект\1.db`

4. ✅ Обновлен React компонент для отображения данных

## Доступные API endpoints:

- `GET /api/tables` - получить список всех таблиц в базе данных
- `GET /api/data/{tableName}` - получить данные из указанной таблицы

## Как использовать:

### 1. Запустите ASP.NET Core API:

```bash
cd C:\ReactAspNetCoreProject
dotnet run
```

### 2. Запустите React приложение:

```bash
cd C:\ReactAspNetCoreProject\clientapp
npm start
```

### 3. Откройте браузер:

Перейдите на http://localhost:3000

### 4. Выберите таблицу:

На странице появится выпадающий список с доступными таблицами из вашей базы данных. Выберите таблицу, и её данные отобразятся в таблице ниже.

## Если база данных не найдена:

Если файл базы данных находится в другом месте, измените путь в файле `appsettings.json`:

```json
{
  "ConnectionStrings": {
    "DefaultConnection": "Data Source=ПУТЬ_К_ВАШЕЙ_БД.db"
  }
}
```

**Важно:** Используйте двойные обратные слэши (`\\`) в пути на Windows.

## Решение проблем:

### Ошибка "База данных не найдена"
- Проверьте путь к файлу базы данных в `appsettings.json`
- Убедитесь, что файл существует
- Проверьте права доступа к файлу

### Ошибка CORS
- Убедитесь, что оба сервера запущены (ASP.NET Core и React)
- Проверьте, что React запущен на порту 3000

### Таблица не отображается
- Проверьте название таблицы (регистр может иметь значение в некоторых случаях)
- Используйте endpoint `/api/tables` чтобы увидеть список доступных таблиц
